## 浮点数

- 标准: IEEE 754
- 表示: (-1)^Sign × 1.Fraction × 2^(Exponent - Bias)
  
| 部分 |	含义 |
|   :-    |  :-  | 
|Sign|	符号位（0 正，1 负）|
|Exponent|	指数（移码表示）|
|Fraction (Mantissa)	|尾数（小数部分）|

- 流程图
X
 ├─► 确定符号位 Sign
 ├─► 转成二进制（整数部分 + 小数部分）
 ├─► 规格化成 1.xxxxx × 2^k
 ├─► k → Exponent（加 Bias）
 └─► xxxxx → Fraction（去掉最高位 1）

- float - 
|1bit|8bit|23bit|


- double

|1bit|11bit|52bit|

### 用示例说话
- 用float `value = -199.285` 转换成 二进制表示

1. 符号位 
   value < 0 ? 1 : 0
2. 整数部分
```
   value1 = 199;

   // 199 / 2 = 99 余 1
   // 99 / 2 = 49 余 1
   // 49 / 2 = 24 余 1
   // 24 / 2 = 12 余 0
   // 12 / 2 = 6 余 0
   // 6 / 2 = 3 余 0
   // 3 / 2 = 1 余 1
   // 1 / 2 = 0 余 1
   value1bits = '11000111'
```

3. 小数部分
```
    value2 = 0.285

    0.285 * 2 = 0.57 -> 0
    0.57 * 2 = 1.14 -> 1
    0.14 * 2 = 0.28 -> 0

    0.28 * 2 = 0.56 -> 0
    0.56 * 2 = 1.12 -> 1
    0.12 * 2 = 0.24 -> 0
    0.24 * 2 = 0.48 -> 0
    0.48 * 2 = 0.96 -> 0

    0.96 * 2 = 1.92 -> 1
    0.92 * 2 = 1.84 -> 1
    0.84 * 2 = 1.68 -> 1
    0.68 * 2 = 1.36 -> 1
    0.36 * 2 = 0.72 -> 0

    0.72 * 2 = 1.44 -> 1
    0.44 * 2 = 0.88 -> 0
    0.88 * 2 = 1.76 -> 1
    0.76 * 2 = 1.52 -> 1
    0.52 * 2 = 1.04 -> 1

    0.04 * 2 = 0.08 -> 0
    0.08 * 2 = 0.16 -> 0
    0.16 * 2 = 0.32 -> 0
    0.32 * 2 = 0.64 -> 0
    0.64 * 2 = 1.28 -> 1
    0.28 * 2 = 0.56 -> 0 // 开始无限循环

    // 无限循环          01000 11110 10111 00001
    value2bits = "0.010 01000 11110 10111 00001"
```

4. 规范化
```
    // 二进制
    110  00111. 010 01000 11110 10111 00001

    // 标准化
    1.10 00111  010 01000 11110 10111 00001 * 2^7
    //10 00111  010 01000 11110 110
    // k = 7
```
5. 指数位
```
    Exponent = 7 + 127 = 134

    Exponent bits = 10000110
```

6. 最终二进制位
```
1| 10000110 | 10 00111  010 01000 11110 110
```
